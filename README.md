## Development

### Local Preview
```bash
# Open in browser directly
open index.html

# Or use a local server
python -m http.server 8000
# Visit http://localhost:8000
```

### File Structure
```
.
├── index.html      # Main portfolio (HTML + CSS + JS)
├── CNAME          # GitHub Pages domain mapping
├── LICENSE        # License
└── README.md      # This file
```

## Customization

Edit `index.html` to update:
- **Hero section** — Name, tagline, stats
- **Experience** — Work history and achievements
- **Projects** — Portfolio projects with links
- **Skills** — Technical capabilities and proficiency
- **Contact** — Social links and contact info
- **CSS variables** — Colors, fonts, spacing

## Deployment

### GitHub Pages
```bash
# Push to GitHub
git add .
git commit -m "Update portfolio"
git push origin main
```

Portfolio will auto-publish from the `main` branch.

### Custom Domain
Update DNS CNAME record to point to GitHub Pages, then add domain in repository settings.
